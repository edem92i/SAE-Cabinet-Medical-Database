# Projet SAE SQL - Gestion Cabinet Médical

## Description
Ce projet consiste à modéliser et implémenter une base de données pour la gestion d’un cabinet médical.

## Technologies
- SQL
- Modélisation MEA
- GitHub

## Entités principales
- Patient
- Médecin
- Consultation
- Rendez-vous
- Traitement
- Service
- Chambre

## Fonctionnalités
- Enregistrer une consultation
- Gérer les patients
- Gérer les médecins
- Lister les consultations
